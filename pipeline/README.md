# PedsIQ Pipeline

Modern data pipeline for PedsIQ — from KUHS PDF ingestion to knowledge graph analysis.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

## Commands

```bash
# Parse a single PDF
pedsiq parse exam_2024.pdf --output parsed.json

# Validate parsing
pedsiq validate exam_2024.pdf

# Classify questions
pedsiq classify parsed.json --output classified.json

# Build knowledge graph
pedsiq graph --output graph.json

# Run full pipeline
pedsiq run /path/to/pdfs/ --output output/
```

## Architecture

- `extract/` — PDF parsing with sub-part extraction and mark allocation parsing
- `classify/` — Multi-dimensional taxonomy (Format × Bloom's × Content Depth)
- `graph/` — Pediatric knowledge graph (self-curated, no UMLS)
- `analyze/` — Pattern analysis and backtesting

## Web Data Outputs

The current web app consumes:

- `web/src/data/questions.json` — 409 KUHS PYQs
- `web/src/data/metadata.json` — 24 paper metadata records
- `web/src/data/mcqs*.json` — manually curated MCQ practice data
- `web/src/app/structured-answers/topics.ts` — manually curated structured-answer topics

## Arcade Data

Arcade game data is manually curated static JSON in `web/src/app/arcade/[game]/data/`. It is not generated by this pipeline.

Current arcade data sets:

- Dose Duel: dosing questions
- Dose Sniper: falling-card dose rounds
- Feature Wars: differential-diagnosis battles
- Protocol Builder: management protocol steps
- Trap Defuser: examiner trap/correct cards

Any new arcade content should also update `NEUROSCIENCE.md` with the learning mechanism it is meant to exercise.
